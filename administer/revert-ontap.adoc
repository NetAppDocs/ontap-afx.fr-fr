= La restauration ONTAP n'est pas prise en charge avec les systèmes de stockage AFX.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
La restauration d'un cluster ONTAP consiste à transférer tous les nœuds vers la version majeure précédente ONTAP .

Les systèmes de stockage NetApp AFX ne prennent pas en charge la restauration ONTAP . Toute tentative de restauration avec AFX peut entraîner une instabilité du cluster et une perte de données. Vous ne devez pas tenter d'effectuer une opération de restauration sur un système AFX.
